# Software Team RACI Template

**Purpose:**  
Clarify responsibilities across roles to prevent overlaps and gaps. Use this template to define who is **Responsible (R)**, **Accountable (A)**, **Consulted (C)**, and **Informed (I)** for key tasks or deliverables.

---

| Task / Deliverable                          | Core Engineer | Platform Engineer | Tech Lead | TPM | Architect | SRE |
|---------------------------------------------|---------------|-----------------|-----------|-----|-----------|-----|
| Define feature requirements                 | R             | C               | A/C       | I   | C         | I   |
| Implement feature code                       | R             | C               | A/C       | I   | C         | I   |
| Code review / PR approvals                   | R/C           | R/C             | A         | I   | C         | I   |
| Testing / quality assurance                  | R             | R/C             | A/C       | I   | C         | I   |
| Platform tool / service delivery             | C             | R/A             | C         | I   | C         | I   |
| Platform enablement for other teams         | I             | R/A             | C         | I   | C         | I   |
| Architecture design decisions                | C             | C               | C         | I   | A         | I   |
| Architecture design communication            | C             | C               | C         | I   | A         | I   |
| Incident response / reliability             | I             | C               | I         | I   | C         | R/A |
| Automation and toil reduction               | I             | R/C             | C         | I   | C         | R/A |
| Career development mentoring / coaching     | I             | C               | R         | I   | C         | I   |
| Program planning / roadmap alignment        | I             | C               | C         | R/A | C         | I   |
| Stakeholder communications                  | I             | C               | C         | R/A | C         | I   |

---

**RACI Legend:**

- **R** = Responsible → Does the work / executes the task  
- **A** = Accountable → Ultimately owns the task / decision  
- **C** = Consulted → Provides input / expertise  
- **I** = Informed → Kept up-to-date on progress  

---

*Tip: Adjust the tasks and roles based on your organization. Consider reviewing this RACI quarterly or whenever team structures change.*
