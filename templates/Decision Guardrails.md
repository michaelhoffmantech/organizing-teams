# Decision Guardrails

## Decision Categories
| Decision Type | Owned by Team? | Collaboration Required | Escalation Path |
|---------------|----------------|----------------------|----------------|
| Example: API design | ✅ | ❌ | - |
| Example: Major infra changes | ❌ | ✅ | CTO |

## Guidelines
- Define thresholds for escalation
- Include pre-approved options for common decisions
- Include risk boundaries
