# Collaboration Modes Matrix

| Interaction Type | Frequency | Channel | Synchronous / Asynchronous | Commitment Required |
|-----------------|-----------|--------|----------------------------|------------------|
| Example: Planning | Weekly | Zoom | Sync | 100% |
| Example: API discussion | As needed | Slack | Async | Optional |

## Notes
- Define clear expectations for each mode
- Timebox collaboration where possible
- Include escalation rules
